# Currency Converter

A simple currency converter built with HTML, CSS, and JavaScript that fetches real-time exchange rates using the ExchangeRate API. Users can select currencies, enter an amount, and instantly view the converted value.

## Features

- Real-time currency conversion
- Supports multiple international currencies
- Displays exchange rate information
- Responsive user interface
- Error handling for failed API requests

## Tech Stack

- HTML5
- CSS3
- JavaScript
- ExchangeRate API
